## Руководство пользователя

Запустите программу.

![Запущенная программа](https://github.com/bonny2155-dotcom/demexam/blob/main/img/1.png)

Нажмите на кнопку авторизация и введите данные в открывшемся окне.

![Окно авторизации с введенными данными для входа пользователя](https://github.com/bonny2155-dotcom/demexam/blob/main/img/2.png)

Нажмите кнопку войти.

![Окно после нажатия на кнопку входа при правильном вводе данных](https://github.com/bonny2155-dotcom/demexam/blob/main/img/3.png)

Еще раз нажмите на кнопку входа в следующем открывшемся окне.

![Окно после нажатия на кнопку входа в открывшемся окне](https://github.com/bonny2155-dotcom/demexam/blob/main/img/4.png)

При нажатии кнопки удалить, пользователь не будет удален так как нельзя удалить самого себя.

![Окно после нажатия на кнопку удаления](https://github.com/bonny2155-dotcom/demexam/blob/main/img/5.png)

При нажатии на кнопку изменить откроется окно редактирования в котором пользователь
 может изменить свой логин и пароль.
 
![Окно редактирования и сообщение после успешного редактировая данных](https://github.com/bonny2155-dotcom/demexam/blob/main/img/7.png)
