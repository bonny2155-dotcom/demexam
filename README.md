# Итоговое задание
## Описание:
Проект создан при выполнении подготовки к дем. экзамену для выполнения итогового задания.
В проекте осуществляется авторизация пользователей под разными ролями, работа с ограничением ролей, и базой данных.

## Руководство пользователя
1. Запустите программу.
![Запущенная программа](img/1.png)
2. Нажмите на кнопку авторизация и введите данные в открывшемся окне.
![Окно авторизации с введенными данными для входа пользователя](img/2.png)
3. Нажмите кнопку войти.
![Окно после нажатия на кнопку входа при правильном вводе данных](img/3.png)
4.Еще раз нажмите на кнопку входа в следующем открывшемся окне.
![Окно после нажатия на кнопку входа в открывшемся окне](img/4.png)
5. При нажатии кнопки удалить, пользователь не будет удален так как нельзя удалить самого себя.
![Окно после нажатия на кнопку удаления](img/5.png)
6. При нажатии на кнопку изменить откроется окно редактирования в котором пользователь может изменить свой логин и пароль.
![Окно редактирования и сообщение после успешного редактировая данных](img/7.png)
